# 자동차 경주 게임

## 진행 방법

* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정

* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록

* 게임 기능
    * 자동차 기능
        * 숫자가 4이상이면 이동
        * 숫자가 3이하이면 멈춤
        * 0~9 사이 랜덤 값 생성
    * 자동차 현재 이동 거리 출력
    * 우승 자동차 이름 출력
* 사용자 기능
    * 자동차 이름 입력
        * 이름은 5자 이하 제한
    * 자동자 이동 횟수 입력
